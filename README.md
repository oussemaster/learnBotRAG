# learnBotRAG 🧠📚

Ein lokales, KI-gestütztes RAG-System (Retrieval-Augmented Generation) zum Abfragen von eigenen Uni-Skripten, Zusammenfassungen und CSV-Katalogen.

## 🚀 Features
* **Inkrementelles Laden:** Neue Dokumente werden der Datenbank hinzugefügt, ohne alles neu zu berechnen.
* **Auto-Cleanup:** Verwaiste Dateien (gelöschte PDFs) werden automatisch aus der Datenbank entfernt.
* **Interaktiver Chat:** Direktes Fragen-Stellen im Terminal.
